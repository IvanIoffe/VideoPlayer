# VideoPlayer - приложение для просмотра видео

### Стек:
- **Language - Kotlin**<br />
- **UI - XML**<br />
- **Architecture - Clean Architecture, MVVM**<br />
- **DI - Dagger/Hilt**<br />
- **Network - Retrofit**<br />
- **Storage - Room**<br />
- **Annotation processor - ksp**<br />
- **Serialization - KotlinXSerialization**<br />
- **Async - Coroutine/Flow**<br />
- **VideoPlayer - ExoPlayer**<br />
- **LoadImage - Coil**<br />
